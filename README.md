# Nazm

Nazm is a modern, cross-platform To-Do application designed to help you organize your daily tasks efficiently and intuitively. With a focus on speed, reliability, and a seamless user experience, Nazm empowers users to manage their tasks, set priorities, and track progress—anytime, anywhere.

## Features

- **Task Management:** Add, edit, delete, and mark tasks as completed.
- **Task Prioritization:** Assign priorities to tasks to focus on what matters most.
- **Scheduling:** Set due dates and reminders to stay on track.
- **Offline Support:** Access and manage your tasks even without an internet connection.
- **Cross-Platform:** Available on Android, iOS, Windows, macOS, and Linux.
- **Light & Dark Themes:** Switch between light and dark modes for comfortable viewing.
- **User Authentication:** Secure sign-up and login for personalized task management.
- **Data Synchronization:** Sync your tasks across devices (if cloud integration is enabled).

## Tools & Technologies Used

- **Flutter:** The main framework for building a beautiful, responsive, and cross-platform UI.
- **Hive:** A lightweight, fast, and secure NoSQL database for local data storage, ensuring your tasks are always available offline.
- **Firebase:** Used for user authentication and optional cloud data backup and synchronization.
- **Provider/Bloc:** For efficient and scalable state management.
- **Google Fonts:** For enhanced typography and a modern look.
- **Internet Connection Checker:** To handle online and offline scenarios gracefully.
- **Adaptive Theme:** For dynamic switching between light and dark modes.
